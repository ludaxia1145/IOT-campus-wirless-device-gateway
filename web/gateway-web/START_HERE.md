# 👋 校园网关管理系统 - 从这里开始

欢迎使用校园网关管理系统Web前端！本文件将指导你快速上手。

## 🚀 立即开始（3步）

### 第1步：安装依赖（首次需要）
```bash
npm install
```

### 第2步：启动应用
**Windows 用户**（推荐）：双击 `start-dev.bat`

或

**命令行启动**：
```bash
npm run dev
```

### 第3步：打开浏览器
访问 `http://localhost:5173`

完成！🎉

## ❓ 常见问题

**Q: 应用无法连接到网关**
A: 查看 `GATEWAY_CONFIG.md` 中的 "故障排除" 部分

**Q: 如何修改网关IP地址**
A: 编辑 `src/api/gateway.js` 第5行的 `GATEWAY_URL` 变量

**Q: 生产版本如何部署**
A: 查看 `README.md` 中的 "部署方法" 部分，有4种方式供选择

**Q: 如何添加新的功能模块**
A: 查看 `README.md` 中的 "开发建议" 部分

**Q: 项目默认端口是多少**
A: 开发环境为 5173，可通过 `npm run dev -- --port 3000` 修改

## 💡 提示

- **首次启动较慢** 是正常的（需要初始化依赖）
- **修改代码后自动刷新** 是Vite的HMR功能
- **生产版本比开发版本快得多** 是因为优化和压缩
- **生产版本大约 1.4MB** 主要是第三方库
